Connect with Microsoft Teams

To enable sending alert notifications from CubeAPM to Microsoft Teams, you need to create an Integration within your Microsoft Teams. The following steps guide you through the process:

  1. Create a channel for CubeAPM alert in your Microsoft Teams org.

  2. Right Click on CubeAPM alert channel & select Workflows.

  3. Go to Workflows Home and select + Build from scratch.

  4. In the trigger search box type:

    When a Teams webhook request is received

  5. Select that trigger -> for "Who can trigger the flow" choose: Anyone and click + Next Step.

  6. Search for Post card in chat or channel action.

  7. Fill the required fields in Post card in chat or channel action:

     Post as        -> Flow bot
     Post in        -> Channel
     Team           -> <your team>
     Channel        -> <cubeapm-alert-channel>
    1. In the Adaptive Card -> When you click in Adaptive Card section there's pop-up comes select Expression and paste this json:
    json(string(triggerBody()?['attachments'][0]['content']))

  8. After configuring these Setup Click on Save and it will show in the WorkFlows Tab.

  9. Select the workflow and click on Copy Webhook Link.

  10. After copying the Workflow URL go to CubeAPM Alert Page -> Configure Alert -> Receiver Type select Microsoft Teams -> Webhook URL paste the copy webhook link -> click Done.
